Je m'appelle Clarel Grondin. Je suis développeur web freelance basé à Saint-Denis, La Réunion. Formé en électronique avant de plonger dans le développement d'applications, j'ai un profil un peu atypique : je comprends autant ce qui se passe à l'écran que ce qui se passe derrière les machines.

Expérience interactive 2025

Drive —
Mini open world 3D

Conduisez une Honda Civic EK dans une ville de nuit générée procéduralement. Un terrain de jeu technique : rendu 3D temps réel, physique voiture, minimap et contrôles mobiles.

L'idée

Explorer les capacités de rendu 3D dans le navigateur sans moteur tiers — uniquement Three.js, du code géométrique et une physique de voiture artisanale.

Ce que j'ai construit

Génération procédurale de la ville, système de collision par AABB, caméra follow lissée, minimap précalculée et compatibilité tactile complète.

Particularité

100 % statique, zéro dépendance serveur. Tourne directement dans le navigateur, déployé sur GitHub Pages sans backend.
